# 새로운 형태의 CSS 로딩 애니메이션(위 아래로 스무스하게 움직임 효과)

## 영상을 보기에 앞서 생각해보기

- `translateY`와 `animation-delay`를 사용할 것으로 예상
- 각 각의 요소마다 `animation-delay`를 다르게 주면 구현가능할듯?

## 알고 있는 것 & 새롭게 알게 된 것

- 일단 예상했던 그대로 구현을 했다.
- `border-radius: 50%`로 하면 완전한 원형을 만들 수 있다.
- 각 요소에 `@keyframe`을 넣는다. 2초동안 실행되고 실행 횟수는 무한하다.
- `transparent`는 요소를 상속받는다.
- `transform: translateY()`를 사용하여 위 아래로 요소를 움직인다. 음수 값을 가지면 위로 이동한다. 위로 이동했다가 다시 원점으로 되돌린다.
- 여기에서 중요한 것은 동일한 애니메이션 효과를 주되 각 요소마다 애니메이션이 실행되는 시점을 다르게 하는 것이다. 각 요소마다 `animation-delay`를 별도로 주자.
