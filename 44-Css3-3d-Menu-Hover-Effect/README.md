# 3D 호버 메뉴 효과

## 영상을 보기에 앞서 생각해보기

- 이 예제는 직접 한번 해보고 싶어서 `challenge.html`에서 실습해보았다. 성공~

- 최대한 태그를 작성하지 않고 문제를 해결하고 싶어서 가상 요소와 `attr`을 사용하였다. 가상 요소의 content 내용과 가상 요소의 대상 요소의 내용이 같아야하는데 CSS에서 직접 content 내용을 지정해주는 것보단 태그의 `title` 속성으로 지정하는게 더 유지보수하기 좋을 것이라고 생각했다. `attr(title)`이라고 하면 `title` 속성 값을 content 내용으로 사용한다.

- 추가적으로 처음에는 원리 파악은 되었는데 `hover`시 영상과 같은 3D 효과가 안나서 고민이였는데. 요소의 자식이 3D 공간에 배치하도록하는 `transform-style: preserve-3d`를 사용하면 된다.

## 알고 있는 것 & 새롭게 알게 된 것

- 내가 실습해본것과 비슷하면서도 달랐다. 일단 가상 요소 `::after, ::before`를 모두 사용했고 `translateY()`를 사용하였다.

- 주로 사용된 속성은 대부분 비슷했다. 핵심은 `transform-origin`, `transform-style`, `rotateX()` 이 3가지가 핵심이다. 실습과 다른 점이라면 나는 요소 전체를 rotateX로 회전시켰는데 예제에서는 가상 요소 각 각에 속성을 주고 값을 조작했다.

- 꾸준히 하다보니 간단한 CSS 애니메이션에 대한 감을 잡고 있는 듯하다.
