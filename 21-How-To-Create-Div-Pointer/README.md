# DIV 포인터

## 영상을 보기에 앞서 생각해보기

- 앞서 만든 예제와 동일하나 입체적인 효과가 없는 예제이다.
- 해당 예제의 핵심은 말풍선 효과를 위해 아래 화살표 모양을 만드는게 핵심이며 이는 가상 요소(`::after, ::before`)를 사용하여 구현할 수 있다. 그리고 구현시에 크게 두가지 방법이 있을 수 있다.
  1. width, height 설정 후 rotate로 회전하고 화살표 모양에 해당하는 곳에만 border 값 주기
  2. border로 삼각형 만들기 `border-left: 1px solid transparent`가 핵심(border로 삼각형 만들기라고 구글링하면 나온다)
