# CSS 호버 아이콘

## 영상을 보기에 앞서 생각해보기

- 가상 요소로 요소 중앙에 보이지 않는 둥근 점을 만들고 마우스 호버시 그 점이 커지는 방식으로 하면 될듯하다.

## 알고 있는 것 & 새롭게 알게 된 것

- 아이디어는 동일하나 점을 만드는 방식에서 차이가 있다. 나는 `width, height`로 만든 요소를 크게 만들 생각이였는데. `scale`로 효과를 나타내었다. `scale(0) -> scale(1)` 애니메이션 효과가 어떻게 저렇게 되는걸까? 잠깐 고민해봤는데. 생각해보니 `transform-orgin`의 기본값이 `center, center`이니까 `가로/세로` 중앙에서 커지는 모양이 되기에 가능한거였다. 그렇구나~
