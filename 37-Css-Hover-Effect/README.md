# CSS 호버 효과

## 영상을 보기에 앞서 생각해보기

- 가상 요소로 영역을 만들고 `position: absolute; width: 100%; height: 100%`로 부모의 크기하게 준다. 그 다음 `left: -100%`으로 영역을 숨긴 다음 `hover`시 `left: 0`으로 주면 영상에 나오는 효과처럼 될 것으로 예상한다.

## 알고 있는 것 & 새롭게 알게 된 것

- 내가 예상한 것과 정확히 일치했다!
- 계속하다보면 애니메이션 효과를 충분히 쉽게 유추가능할 듯 하다.
- 그리고 가상 요소는 `:`가 아니고 `::`이다. `:`로 해도 호환성 유지로 인해 작동은 하지만 명세상에는 `::`이다!
