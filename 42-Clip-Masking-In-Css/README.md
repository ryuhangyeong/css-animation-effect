## 알고 있는 것 & 새롭게 알게 된 것

- `background: url()` 배경 이미지 설정
- `background-size: cover` 배경 이미지를 화면에 맞게
- `background-attachment: fixed` 배경 이미지를 뷰포트 내에서 고정할지, 아니면 자신의 컨테이닝 블록과 함께 스크롤할지 지정합니다.
- `-webkit-text-fill-color: transparent` (비표준) 배경에 깔아놓은 gradient 혹은 이미지를 보여주기 위해 투명도 처리
- `background-clip: text` css 로 텍스트에 gradient, 이미지 적용하기
  텍스트 내부가 `gradient` 혹은 `이미지`로 채워진다.
