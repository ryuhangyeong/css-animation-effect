## 알고 있는 것 & 새롭게 알게 된 것

- 처음에 비슷한 마크업으로 시작하길래 깜짝 놀랐다. 거의 한달 넘게 CSS 효과를 실습하고 고민하다보니 생각의 흐름도 비슷해졌나보다.
- 튕겨지는 공을 보면 절반을 기준으로 서로 색상이 다른걸 알수 있다. 나는 이 부분을 가상 요소로 만들어주었다. 일단 전체 사각형을 `border-radius`로 원형을 만들고 기존 크기의 절반에 해당하는 가상요소를 만들어서 덮어씌우는 방식을 채택하였다. 그런데 영상에서 구현하는 방법은 획기적이다!
- width로 크기를 가지게 하고 `border-top, border-bottom`을 이용하여 각 각 요소 절반을 표현한다. 그 다음에 `border-radius`를 주면 절반씩 색상이 다른 원형이 만들어진다. 아 그러면 이런 방법으로 절반 원형도 표현이 가능하겠구나.. 라는 생각을 했다.
- 내가 시도한 것과 비교해본다면 아이디어는 비슷하나 영상의 구현이 더 깔끔하다. 구현하면서 몇가지 애매한 부분이 있었는데 영상을 보면서 이렇게 구현하면 내가 했던 고민을 굳이 안해줄 수 있었구나.. 라는 걸 알게 되었다. 하나 예를 들자면 그림자를 중앙에 위치시키기 위해 불필요한 코드를 나열했는데 그럴 필요없이 절대값 기준으로 정렬한 뒤에 scale로 크기만 조절했으면 중앙 정렬을 위해 굳이 불필요한 코드를 작성하지 않아도 되었을 듯 하다.
