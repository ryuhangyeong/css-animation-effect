# 회전하는 원형 애니메이션

## 영상을 보기에 앞서 생각해보기

- 각 요소가 반투명이라 아래 요소가 보인다.
- 각 요소를 하나씩 돌리기보단 감싸는 컨테이너를 돌리는게 더 효율적이지 않을까? 라는 생각을 하였다.

## 알고 있는 것 & 새롭게 알게 된 것

- 예제로 보여준 것과 실제 구현한 영상에는 차이가 있었다. 시작하자마자 자기 자리에서 회전하는게 아니라 각 요소가 특정 시간 대기 후 차례대로 회전하다가 결국에 모두 회전하게 된다.
- `transform-origin: bottom center`를 이용하여 아랫쪽 중간을 기준으로 `360도` 회전한다.
- 각 요소마다 `animation-delay`를 다르게 준다.
- 만약 처음부터 해당 모양을 유지하면서 회전한다면 각 요소마다 `rotate` 값을 주어서 모양을 만든 후 전체를 감싸는 부모 요소를 계속 회전시키면 될 듯하다.
