# 그림자 효과를 사용한 애니메이션

## 영상을 보기에 앞서 생각해보기

- 나름 고민해보고 시간을 들여서 실습해보았는데. 영상에 있는 효과처럼 표현하는게 쉽지 않았다.
- 애니메이션 자체는 요소를 배치한 뒤에 요소를 감싸는 부모를 회전시키면서 요소 하나하나를 바깥쪽에서 안쪽으로 이동시키면 비슷한 모양이 나올줄 알았는데 바깥쪽에서 안쪽으로 이동될때 아무래도 부모 요소 자체가 회전하다보니 회오리마냥 휘어서 안쪽으로 밀려들어갔다. 음..
- 영상보면서 공부하자.

## 알고 있는 것 & 새롭게 알게 된 것

- 오.. 사실 정말 생각치도 못한 방법이다. 나는 요소 갯수만큼 아이템을 직접 마크업으로 정해주었는데 요소 하나하나가 box-shadow 였다니!
- 예전에도 이런 원리를 이용한 효과가 있었던거 같긴 하다. 기억해야하는 것은 `box-shadow`의 세번째 인자는 그림자 번짐 강도이다. 이것은 0으로 하면 요소 하나가 복사되는 효과를 가진다. 원본 요소를 기준으로 `x, y` 떨어진 곳에 동일 원소를 위치시킬 수 있다!
- 나는 바깥쪽에서 안쪽으로 밀려들어가는게 `transform: translateY`라고 생각했는데 `box-shadow`로 이게 가능했다. 안쪽으로 들어갈 때는 `box-shadow: 0 0 0 color`이고 바깥쪽으로 나올 때는 원래 그림자 요소이다.(`box-shadow: 30px 30px 0 color`) 이게 생각해보니까 `box-shadow`의 첫번째와 두번째 인자가 원본 요소 기준으로 x,y 축의 위치니까 이렇게 이동할 수 있는거구나.
- 또 이렇게 하나 배워간다! `부모 요소를 기준으로 동일한 요소를 부모 기준으로 배치할 때 box-shadow`를 사용하면 좋을 듯하다.
