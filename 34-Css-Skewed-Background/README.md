# CSS 비스듬한 배경

## 영상을 보기에 앞서 생각해보기

- 제목에서 유추할 수 있듯이 `skew`를 사용할 것이다. 그런데 모양상 요소가 두개가 있어야 영상에 있는 모양을 만들 수 있을 것으로 예상한다.
- 추측해보건데 50%에 해당하는 요소를 하나 만든 다음에 그 요소의 가상 요소를 만드는게 효율적인 방법이 아닐까 싶다.
- `skewY`로 모양을 잡고 `translateY`로 올린다음에 `overflow: hidden`으로 감추면 될 듯하다. 아래 요소도 그냥 `translateY`로 끌어올리면 될 듯하다.

## 알고 있는 것 & 새롭게 알게 된 것

- 음.. 머랄까 내가 위에서 생각해본것은 반은 맞고 반은 틀렸다. 일단 아이디어는 맞았지만 아이디어를 대입하는 대상이 달랐다. 내가 생각했던 것보다 영상의 방식이 좋다. 이렇게하면 굳이 `overflow: hidden`으로 가려줄 필요없으니까. 조금 더 깊게 고민해볼걸.. 그래도 이제 어느정도 애니메이션 효과를 보면 맥락 파악은 되는 듯하다.
