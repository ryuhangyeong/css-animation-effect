# 마우스 오버시 회전하는 텍스트

## 알고 있는 것 & 새롭게 알게 된 것

- 요소 가로/세로 중앙 정렬 방법
- `clearfix` 방법론
  영상에서는 `clearfix` 요소를 추가하여 해결하지만 다른 더 좋은 방법도 존재한다는 것을 이미 알고 있음. 4년 전 영상이라 이렇게 처리한듯? 방법이야 다양하니까.

- 아무런 효과(:hover) 를 주지 않았는데도 `transition` 설정시 특정 속성에서 효과가 생겨났다. `font-size`의 경우 `transition`의 영향을 받는 듯하다.

- `rotate`를 사용하면 회전할 수 있다.
- `transition-delay`를 사용하면 특정 시간 지연 후 애니메이션을 실행할 수 있다. 지연을 통해서 글자가 순서대로 애니메이션 효과가 일어나도록 표현할 수 있다.
