# 마우스 호버시 특정 지점을 기준으로 회전

## 알고 있는 것 & 새롭게 알게 된 것

- `perspective` 원근법을 표현하기 위해서 사용한다.
- `transform-origin`
  CSS transform 속성과 함께 사용되는 속성으로서, 회전 중심(원점·기준점)을 지정합니다. rotate(), skew() 등의 회전, 변형 속성을 사용하기 전에 기준점을 지정해 둡니다. 초기 값은 50% 50%으로 요소의 중심점이 됩니다. transform-origin의 속성 값은 백분율(%)과 키워드 중 하나로 지정할 수 있습니다.
  회전을 하는 경우 특정 기준점에서 회전을 해야하므로 기준점을 설정할 때 사용한다. 값은 x축 / y축이라고 생각하면 된다.
  ```css
  transform-origin: 100% 0; /* x축으로 100% / y축으로 0 이므로 상단*/
  ```
- `rotateX` 회전
  음수는 정면, 양수는 반대
