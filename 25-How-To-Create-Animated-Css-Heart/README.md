## 영상을 보기에 앞서 생각해보기

- 두 개의 영역(A, B)이 존재
- A, B가 함께 rotate를 하고 A는 거기에서 멈추고(둥근 모양으로 바꾸면서) B는 특정 시간 경과 후(delay) rotate로 한 바퀴 돌며 애니메이션 종료
- 마크업이 문제인데 이런 모양의 경우 별도로 요소 두개를 만드는것보단 가상 요소로 만드는게 나은 방법이지 않을까?

## 알고 있는 것 & 새롭게 알게 된 것

- 일단 마크업은 가상 요소를 사용하는게 맞았다.
- 아이디어는 맞았는데 구현 코드를 보면서 많이 배웠다.
- 일단 `transform-origin`으로 rotate하는 기준점을 오른쪽 아래로 설정해준다.
- 자식 요소가 `position: absolute;`이고 `width, height`가 100%이면 부모의 너비와 높이를 모두 차지한다.
- `hover`하는 경우 두 요소 각각의 `rotate`와 `translateY`로 조절하면 된다.
