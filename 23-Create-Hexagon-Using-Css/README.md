# CSS 헥사곤

## 영상을 보기에 앞서 생각해보기

- `clip-path`라는 속성을 알긴 하는데 써볼 일이 많지 않아서 잘 모른다. 근데 아마 `polygon`써서 구현할거 같다? 예전에 CSS 강의에서 비슷한걸 본적이 있긴 하다.

## 알고 있는 것 & 새롭게 알게 된 것

- CSS를 하면서 제일 불편하다고 해야할까? 이해가 안되는게 자식 요소에서 `height: 100%`를 하는 경우 부모 높이를 인식을 못한다는 거다. 물론 직접 부모 높이를 지정하면 가능하지만 말이다. 그런데 이번 예제에서 깨닫게 된게 부모에 `position`이 있으면 자식에서 `position, width, height`를 쓰면 부모를 인식한다는 거다. 원래 잘 알려진 트릭(부모는 `relative`, 자식은 `absolute`)를 알고 있긴 하지만 이 경우에는 부모 높이를 100%로 인식할 수 있다는게 조금 신기한거 같다.

- 이거 머지? 왜 이래 신박하지? 사실 나는 `clip-path`로 그릴 줄 알았는데 이거를 `div` 속성과 가상 요소만으로 만들 생각을 하다니!

- 내부에 있는 선은 `border`로 표현했다.
