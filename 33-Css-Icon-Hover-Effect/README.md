# CSS 호버 아이콘 효과

## 영상을 보기에 앞서 생각해보기

- 마우스 호버시 왼쪽에서 오른쪽으로 슬라이드(여기에서는 width 값을 0에서 요소 크기만큼 늘릴 것 같다)되는 부분은 가상 요소일 것으로 예상한다.
- 가상 요소와 아이콘은 별도의 레이아웃이므로 `z-index`의 변화도 있지 않을까? 라고 추측한다.
- 요소 내부에 있는 아이콘들이 뒤집혔다가 다시 정상으로 돌아오는데 이 부분은 `rotateY` 라고 예상한다. 중간에 아이콘이 커지는데 이건 `scale`일거다.

## 알고 있는 것 & 새롭게 알게 된 것

- 유튜버님이 기존에는 `transform`으로 `container`를 중앙 정렬하셨는데 이제는 `margin: 0 auto`를 사용하신다!
- 내가 생각한 것처럼 왼쪽에서 오른쪽으로 슬라이드되는 부분은 가상요소였다. 하지만 애니메이션 효과에서는 `width`가 아닌 `left`를 사용했다. 근데 이 방법이 더 좋은거 같다. 왜냐하면 조금 더 유지보수하기 좋은 코드이기 때문이다. width 값으로 하면 요소값 변경시 width 값도 변경해주어야하지만 `position: absolute; left: -100%`를 한 자식 요소는 부모 요소의 크기를 인식하기 때문이다. `left`를 마이너스로 했기에 기존 부모 요소의 크기만큼의 요소가 왼쪽에 배치되고 `left`를 0으로 하는 경우 원래 자리로 되돌아오는 방식으로 구현되었다.
- `box-shadow`는 여러개 지정할 수 있다. 여러개 지정하여 3D 효과를 구현하는 예제도 앞서 있었다.
