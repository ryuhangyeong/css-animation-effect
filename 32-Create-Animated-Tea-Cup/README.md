## 알고 있는 것 & 새롭게 알게 된 것

- `컵`은 왼쪽 / 오른쪽 아래 `border-radius`을 각각 주는것 이외에 특이사항 없다.
- `컵 입구`를 어떻게 할까 궁금했었는데 의외로 간단했다. 직사각형을 가상 요소로 만들고 `border-radius: 50%`를 주고 테두리는 하얀색, 배경 색상은 커피 색으로 주고 `컵 입구`를 끌어올렸다. 이때 `transform: translateX(-50%)`로 현재 위치를 기준으로 요소의 절반만큼 올린다.
- `컵 손잡이`도 가상 요소로 비슷하게 만들면 된다. 생각보다 원리가 간단하다.
- `컵 받침대`에서 아래 부분 모양을 어떻게 할수 있을까 했는데 생각보다 간단했다. `border-bottom-left-radius: %`로 하면 저런 모양(?)의 테두리를 만들 수 있었다. 지금까지 `px`로만 설정해보고 %로는 한번도 설정해본적이 없어서 저런 모양이 나올줄은 몰랐다. 신기해~
- `연기`는 막대기를 일렬로 세우고 각 요소마다 애니메이션 효과와 딜레이를 각자 주면 된다. 여기에서 흐린 효과는 `blur`를 사용했다.
