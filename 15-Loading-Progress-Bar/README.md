# 배경 화면이 변경되는 진행 로딩바

## 영상을 보기에 앞서 생각해보기

- 순수 CSS로 구현하려면 `@keyframes`로 구현할 수 밖에 없을 듯하다.
- `%` 단위로 `body`의 `background-color`와 `bar`에 있는 `width / background-color` 값을 변경해주면 되지 않을까?

## 알고 있는 것 & 새롭게 알게 된 것

- 일단 내가 생각한 방식대로 구현하는게 맞다. 눈 여겨보아야 하는 것은 로딩바를 생성하는 방식이다. 가상 요소인 `::after`를 사용하였다.
- `inset`를 사용하여 안쪽 그림자를 표현했다. 앞서 사용했던 속성이라 바로 알 수 있었다~
- 자식의 `height: 100%`인 경우 부모 높이를 따른다. 이때 직접 `height` 값을 지정해주어야 한다. `height: 100%`가 원하는대로 작동하지 않는 경우가 많은데 부모 높이가 지정되어있지 않기 때문이다.
- 가상 요소에 `animation`을 주어 `width / background-color` 값을 변경해준다.
- `body` 요소에도 동일하다. 대신에 `background-color`만 변경되는 `animation`을 주자.
