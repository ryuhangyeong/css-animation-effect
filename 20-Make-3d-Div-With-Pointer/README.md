# 3D 포인터

## 영상을 보기에 앞서 생각해보기

- 이번 예제는 조금 복잡해보인다!
- 일단 `rotateY`를 사용할 것으로 예상
- 입체적인 효과를 주기 위해 그림자 같은게 있는데 이게 `box-shadow`인지 아니면 다른 것인지는 잘 모르겠다.
- 말풍선 모양을 만들기 위해 아래 뾰족한 삼각형이 있는데 이건 아마 가상 요소(`::after, ::before`)를 통해 만들 것으로 예상된다.

## 알고 있는 것 & 새롭게 알게 된 것

- `box-sizing: border-box;`는 width 계산시 padding 값까지 포함되도록 한다.
- `position: absolute`를 하면 요소가 `display: block`의 효과가 있다.
- 뾰족한 삼각형을 만들기 위해 정사각형을 만들고 `rotate`를 해준다음 화살표 모양에 해당하는 영역에만 `border`를 주는 식으로 구현을 했다. 음! 이거 말고도 `border: 1px solid transparent` 등의 trick이 있지만 이걸로 했을 때 개인적으론 크기 조절하는게 조금 힘들더라. 그래서 영상에 나오는 것처럼 하는 것이 더 좋은 방법인 듯하다. 아무래도 직접 `width, height` 값을 정해줄 수 있으니 말이다.
- 말풍선 아래 그림자를 통해 말풍선이 살짝 떠있는 느낌을 줄 수 있는데 이때도 가상 요소를 사용했다. 그리고 `filter: blur(5px);`를 사용하여 흐리게 해주었다.
- 예상과 다르게 `rotate`가 아닌 `skew` 를 사용했다. 음 묘하네..
- 위에서 상자에 입체적 효과가 있다고 했는데 여기는 예상처럼 `box-shadow`를 사용했다. 그리고 색상을 여러개 주고 1px씩 차이를 주어서 그림자가 깊게 쌓인 형태로 구현했다. 그리고 `hover`시 `shew`가 바뀌면서 입체 효과가 생기는 위치가 달라져야하므로 `box-shadow`를 변경해주었다.
