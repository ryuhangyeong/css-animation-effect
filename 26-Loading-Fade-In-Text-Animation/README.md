# 페이드인 텍스트 애니메이션 로딩

## 영상을 보기에 앞서 생각해보기

- 진입 애니메이션은 왼쪽에서 오른쪽으로 비스듬이 투명도 0 -> 1 (이 부분 애니메이션은 정확히 모르겠다)
- 사라질때는 살짝 커지면서(scale) 투명도 1 -> 0
- 각 글자마다 `@keyframes`을 주고 delay를 각각 줄 것으로 예상한다.

## 알고 있는 것 & 새롭게 알게 된 것

- `transform: rotateY(90deg)`로 `left` 축 기준으로 회전하였다. (90도이므로 화면상에서는 안보임, 왼쪽 기준으로 기울였으므로 가려지는 것임) 그리고 `rotateY(0)`이 되면서 글자가 보여지게 한다.
- 더 역동적인 효과를 위해 `translateX(-30px) -> translateX(30px)` 적용
- 각 `li` 요소에 `animation-delay`를 주면서 각 글자가 차례대로 애니메이션 효과가 적용되도록 하였다.
